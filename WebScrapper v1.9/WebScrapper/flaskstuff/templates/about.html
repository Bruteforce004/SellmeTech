ABOUT US



Sell me Tech is a website which provides an easy way to shop for any kinds of tech products online. Since there are many websites available which sell the same product with different prices, "Sell me tech" website serves as a common platform to compare prices and their features. This can therefore save time, provide abundant information about the product and also locate product easily. Each page provides its own features. There is step-by-step instruction to use these features by clicking on 'How To Use'. know more about recent technologies, news articles are also available. This website is created with HTML, CSS and other technologies. This website was developed by four developers as a course requirement mini project. The contact information of the developers is given in the Contact Us' page. Technologies like python and JS in addition to HTML and CSS to serves as the backbone of this project.
