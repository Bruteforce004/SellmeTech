HOW TO USE


  1. Before using our website the user should login.
  2. Login page provides us with two login options:
    •     Through Google account
    •     Through the registered id and password
  3. Once you are logged in, it takes you to the Home page.
  4. The navigation bar leads you to About Us, Contact Us and How To Use pages.
  5. The Home page has a search bar in which you should input your desired mobile model name.
  6. This leads you to the Results page.
  7. The Results page contains 10 results from each websites.
  8. This contains the Name, Image, Price of the model.
  9. Click on the image to lead you to the respective site.
  10. The Home page also has a button which takes you to the page with the latest Tech news.
  11. The news provided here is updated everyday.
  12. One can Log out through the Log out button mentioned on the Login page.
